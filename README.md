# 科研人格测试 · Science-BTI 🔬

一个自娱自乐的「科研版 MBTI」。答 16 道题（每题 7 档量表，可表达"更偏向"或"都有点"），从 4 个维度算出你的科研人格（共 16 型，含 2 款隐藏款），并给每一型贴上梗身份标签：**大佬 / 牛马 / 边缘人 / 摸鱼怪 / 自由派**。玩法与传播方式参考 LBTI / SBTI —— 纯静态单页，可复制邀请链接、生成结果卡图片分享。

计分为**加权累计**：每题在「上句 ←→ 下句」之间选 7 档，档位权重(`data.js` 的 `SCALE`)分别投给两极，按维度累加后取胜出极。结果页会像 MBTI 测试那样展示**每条维度的百分比占比条**（如"理想派 83% / 生存派 17%"）。量表档数可随意增减，只改 `SCALE` 即可，页面圆点数与计分自动适配。

## 4 个维度

| 维度 | 一端 | 另一端 |
|------|------|--------|
| 驱动力 | `P` 理想派 🔥（为热爱） | `S` 生存派 🍚（为毕业/搞钱） |
| 卷 度 | `G` 卷王 🌪️ | `C` 躺派 🛋️ |
| 社 交 | `E` 社牛 🎤 | `L` 独狼 🐺 |
| 产 出 | `D` 深耕 ⛏️ | `Q` 快枪 🏄 |

类型代码即四个字母拼接，例如 `PGED` = 闪耀学术大佬 🌟。16 型全部在 `data.js` 中定义。

## 隐藏款 ✨

`驱动力(P/S)` 与 `产出(D/Q)` 两条轴天然相关——真心热爱的人多半慢慢深耕，为生存的人多半快速出货。因此「理想却追快 `P·Q`」「生存却深耕 `S·D`」这类**反差组合**在现实中较罕见。我们从中钦定 **2 款真·隐藏款**，抽到时结果卡会有金边流光彩带：

| 代码 | 人格 | 反差点 |
|------|------|--------|
| `PGLQ` | 独狼产粮机 🐺 | 理想派里罕见的「独狼 + 快枪」 |
| `SGLD` | 实验室钉子户 🔩 | 生存派里罕见的「闷头 + 深耕」 |

隐藏款完全由 `data.js` 里类型的 `hidden: true` 标记决定，想换成别的型或增减数量，只改标记即可，`app.js` 无需改动。

## 文件结构

```
index.html   页面骨架 + 分享用的 og 标签
styles.css   卡通梗图风样式（配色/圆角/贴纸感）
app.js       状态管理、计分、分享、截图逻辑
data.js      16 道题 + 16 型人格 + 维度定义（改文案只动这里）
```

## 本地预览

ES Module 需要通过 HTTP 访问（不能直接双击 `index.html`）：

```bash
cd research-mbti
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 部署到 GitHub Pages

1. 新建仓库，把 `research-mbti/` 里的文件放到仓库根目录（或 `docs/`）。
2. `Settings → Pages → Build and deployment`：Source 选 `Deploy from a branch`，分支选 `main`，目录选 `/ (root)` 或 `/docs`。
3. 几分钟后即可访问 `https://<你的用户名>.github.io/<仓库名>/`。

也可直接拖到 Vercel / Netlify / Cloudflare Pages，零配置。

## 分享方式

- **复制邀请链接**：结果页「🔗 复制链接」把一句话 + 首页地址复制到剪贴板，发给别人后对方打开是测试首页。
- **结果编码进 URL**：算出结果后地址栏会变成 `?r=PGED`，带这个参数的链接打开会直接展示该结果（方便回访/晒结果）。
- **保存结果卡**：「📸 保存结果卡」用 html2canvas 把结果卡渲染成 PNG 下载，可发朋友圈 / 群。
  html2canvas 从 CDN 加载；若网络不可用会提示「长按结果卡截图」优雅降级。

## 自定义

- 改题目 / 类型文案：编辑 `data.js`（维度字母全局唯一，勿改字母本身，否则代码拼接会乱）。
- 改配色 / 风格：编辑 `styles.css` 顶部 `:root` 的 CSS 变量。
- 数据完整性自检（16 型齐全、match 有效、每维 3 题）：
  ```bash
  node --input-type=module -e 'import {TYPES} from "./data.js"; console.log(Object.keys(TYPES).length)'
  ```

> 纯属娱乐，切勿当真 😄
